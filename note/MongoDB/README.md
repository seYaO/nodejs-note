## 简介


启动MongoDB服务器
```cmd
$ d:
$ cd mongodb\bin
$ mongod -dbpath D:\mongodb\data\db
```

进入shell
```cmd
$ d:
$ cd mongodb\bin
$ mongo
```
```cmd
show dbs  // 显示所有数据的列表

db  // 显示当前数据库对象或集合

use test    // 连接到一个指定的数据库


```
